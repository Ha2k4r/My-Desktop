## Current State

My dotfiles are currently quite basic. Right now this Repo is just a overengineered Archlinux Dotfile install script.

The install script documents itself whilst its being run, and will only run on arch or a OS with pacman. 

## Future

My entire desktop is moving to nixos and will become better documented in the future.

My intention is to create a fundamentally easy to use, and pre-packaged full ecosystem of apps and tools written by me or the opensource community.
